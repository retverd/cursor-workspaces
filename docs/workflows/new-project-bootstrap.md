# Инициализация нового проекта

## Цель

Инициализировать новый репозиторий из шаблона Cursor с минимальным расхождением.

## Шаги

1. Скопируйте `.cursor/`, `docs/` и нужные примеры в целевой репозиторий.
2. Оставьте только те глобальные правила, которые действительно переиспользуемы.
3. Заполните `docs/architecture/overview.md`.
4. Заполните `docs/components/*` для активных компонентов.
5. Проверьте `globs` в локальных правилах.
6. Удалите нерелевантные примеры правил.
7. Добавляйте первые компонентные правила `1x-*` только при необходимости.
8. Зафиксируйте первый коммит, который задаёт контекст проекта.

- Каким частям репозитория нужны собственные локальные правила?
- Какие повторяющиеся типовые ошибки уже известны?
- Какая документация нужна людям и Cursor?
